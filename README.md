# ✨ Sales Forecasting Dashboard

An interactive sales forecasting and demand prediction dashboard built using Streamlit, Machine Learning, and Data Analytics.

## 🚀 Features
- Interactive KPI dashboard
- Sales trend analysis
- Future sales forecasting
- Monthly sales insights
- Dynamic filters
- Gen Z inspired modern UI

## 🛠 Tools & Technologies
- Python
- Streamlit
- Pandas
- Plotly
- Prophet
- Machine Learning
- Google Colab used for data preprocessing, EDA, and model building

## 📊 Project Type
Time Series Forecasting & Business Intelligence Dashboard
